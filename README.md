# Sentimental-analysis-
A lightweight, ready-to-run Sentiment Analysis project built with Python, Scikit-learn, and Streamlit.
This app analyzes customer reviews from the Reviews.csv dataset and predicts whether the sentiment is Positive, Neutral, or Negative.

 Key Features:

 Machine-learning model trained using Scikit-learn

 Fully offline working version

 Clean and organized project structure
